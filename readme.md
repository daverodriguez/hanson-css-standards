# Hanson CSS Standards

These five example files outline the principles of how we write stylesheets at Hanson.
They will serve as a living reference and a starting point for our best practices.

## Files:

1. [Module Structure](src/scss/1-structure.scss)
2. [BEM Syntax](src/scss/2-bem.scss)

6. [View styles](src/scss/6-views.scss)

## Helpful templates

The `resources/templates` directory contains text snippets for common patterns including
media query mixins and table of contents headings. Read the instructions for your IDE for
information on how to use the templates.

### WebStorm/IntelliJ

On **Windows**, copy the templates in `resources/templates/WebStorm` into your `%userprofile%\.WebStorm<version-number>\config\templates` directory.

On **Mac**, copy the templates into `~/Library/Preferences/WebStorm<version-number>/templates`.

### Sublime Text 2/3

On **Windows**, copy the templates in `resources/templates/SublimeText` into your `%APPDATA%/Sublime Text 2/Packages/Sublime Text 2 Snippets` directory.

On **Mac**, copy the templates into `~/Library/Application Support/Sublime Text 2/Packages/User`.
